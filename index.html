Setup up meowto links:

TL; DR: Add a custom search engine with the url http://meowni.ca/meowto/?%s

  • chrome set it up in chrome://settings/searchEngines
  • firefox add a custom keyword
  • safari i don't know yet. send me a pull request if you do!

    Things you can do:

    If you have a custom search engine set up as meowto, type any of these in your search bar:

    • meowto setup
    • meowto help
    • meowto list
    • meowto <alias>
    • meowto add <alias> -> <some-url.com>
    • meowto nuke <alias>

    If you don't, use http://meowni.ca/meowto/? instead

    • http://meowni.ca/meowto/?setup
    • http://meowni.ca/meowto/?help
    • http://meowni.ca/meowto/? list
    • http://meowni.ca/meowto/?<alias>
    • http://meowni.ca/meowto/?add <alias> -> <some-url.com>
    • http://meowni.ca/meowto/?nuke <alias>
